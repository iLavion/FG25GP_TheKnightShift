# FG25GP_TheKnightShift

Top-down spellcasting built in Unity 6000.0.63f1 for the FG25GP assignment.

## Requirements

- [Unity Editor 6000.0.63f1](https://unity.com/download)
- Windows, macOS, or Linux
- (Optional) [Visual Studio](https://visualstudio.microsoft.com/vs/) or [JetBrains Rider](https://www.jetbrains.com/rider/)

## Getting Started

1. Clone the repo: `git clone https://github.com/iLavion/FG25GP_TheKnightShift.git`
2. Open the project folder in Unity Hub.
3. Load `Assets/Scenes/Assignment.unity`.
4. Press Play to start or do whatever ig.
